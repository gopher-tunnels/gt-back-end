# Gopher Tunnels Back-End

Welcome to the official Gopher Tunnels back-end repository!

## Environment setup

### 1. Clone the repository

```bash
git clone
```

### 2. Install dependencies

Yarn:

```bash
yarn
```

npm:

```bash
npm install
```

### 3. Run the dev environment

Yarn:

```bash
yarn dev
```

npm:

```bash
npm run dev
```
